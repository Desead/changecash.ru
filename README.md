Полностью готовый и рабочий вариант ручного обменника

Запуск:

1. Установите на сервер. Достаточно самого простого впс за 500р и последней ubuntu.
2. БД достаточно SQLite. Можно взять готовую. Доступы к базе. Логин: alx пароль 1. Тогда миграции и суперпользователя можно не создавать. API ключи к рапире также не нужны.
3. После установки запустите миграции
4. Создайте суперпользователя
5. Зайдите в админку по адресу ваш_домен/admin и создайте мерчанта
6. Выделите мерчанта и в выпадающем списки сверху, создайте для него все монеты
7. Зайдите в раздел все монеты и отключить те монеты которые вам не интересны
8. Курсы монет подтянутся сами через минуту и автоматически будут обновляться раз в минуту
9. Зарабатывайте. С вопросами можно сюда: https://t.me/oreshkinalexey
10. Ещё знания про обменники: https://samodengi.ru/
