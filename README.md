Полностью готовый и рабочий вариант ручного обменника

Запуск:

1. Установите на сервер. Достаточно самого простого впс за 500р и последней ubuntu.
2. БД достаточно SQLite. 
3. API ключи к рапире также не нужны.
4. Whitebit чисто технический провайдер. Апи ключи ему нужны. Его задача просто генерировать адрес для приёма и всё.
5. После установки запустите миграции
6. 6Создайте суперпользователя
7. Зайдите в админку по адресу ваш_домен/admin и создайте мерчанта
8. Выделите мерчанта и в выпадающем списки сверху, создайте для него все монеты
9. Зайдите в раздел все монеты и отключить те монеты которые вам не интересны
10. Курсы монет подтянутся сами через минуту и автоматически будут обновляться раз в минуту
9. Реализована простая партнёрка. % считается со входящей суммы
10. Есть генерация XML файла. Учитывая наличие партнёрки, то можно полноценно выходить на мониторинги.
